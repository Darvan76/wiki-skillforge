# Roadmap

## v1.1.0 — Polish & QoL

- [ ] **Skill Search** — Search/filter skills in the tree menu
- [ ] **Build Planner** — Allow planning skill builds before committing points
- [ ] **Favorites** — Pin frequently-used skills
- [ ] **Keybind System** — Bind active abilities to custom keys
- [ ] **Tooltip Enhancement** — Show exact stat changes when hovering nodes
- [ ] **Admin Quick Actions** — Common actions with one click from admin panel
- [ ] **Export All Trees** — Batch export functionality

## v1.2.0 — Customization

- [ ] **Custom Effect Types** — Developer API to register new effect types
- [ ] **Custom Particle Effects** — Per-tree particle configuration
- [ ] **Per-Tree Sounds** — Different unlock/upgrade sounds per tree
- [ ] **Premium Tree Themes** — Visual themes for the skill menu
- [ ] **Tree Backgrounds** — Custom background images per tree
- [ ] **Job-Specific Tree Layouts** — Different layouts per job

## v1.3.0 — Social & Competition

- [ ] **Skill Leaderboards** — Server-wide skill level rankings
- [ ] **Skill Comparison** — Compare your tree with other players
- [ ] **Tree Share Codes** — Share builds with codes
- [ ] **Guild/Faction Trees** — Shared progression for groups
- [ ] **Weekly Challenges** — Time-limited XP bonuses
- [ ] **Skill Reset Tokens** — Item-based reset (via DarkRP items)

## v2.0.0 — Major Update

- [ ] **Web Panel** — External web panel for player tree management
- [ ] **REST API** — HTTP API for external tools
- [ ] **Tree Templates** — Community tree marketplace
- [ ] **Auto-Balancing** — Analytics-driven skill rebalancing
- [ ] **Replay System** — View skill progression over time
- [ ] **Event System** — Trigger server events on milestones
- [ ] **MySQL Support** — Optional MySQL backend
